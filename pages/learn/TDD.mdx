## Gherkin

| Syntax     | Meaning                   |
| ---------- | ------------------------- |
| `Feature`  | Describe a feature        |
| `Scenario` | Describe a scenario       |
| `Given`    | Describe a pre-condition  |
| `When`     | Describe an action        |
| `Then`     | Describe a post-condition |
